# Scrobblefy Playlist Maker

Welcome to Scrobblefy! This is a Python-coded program that uses listening data pulled from Last.fm and allows you to create Spotify playlists based on top listened songs.

## Version 0.9.0
### Changelog
This contains all the (hopefully) functioning base code for the project, including Last.fm and Spotify functionality and several sorting and playlist modification options. There is still some styling to be done, hence 0.9 and not 1.0, and any bugs, minor fixes, or otherwise implementation changes that happen before styling will also fall under the 0.9.x umbrella.
### Features
- Last.fm listening data retrieval, including top songs from default and custom timeframes as well as recent listens
- Sorting options for both Last.fm data lists and playlists built from the data
- Full functionality for Spotify exports, including new playlist creation and addition to existing playlists
- Help window to hopefully assist with any intricacies or questions users may have
